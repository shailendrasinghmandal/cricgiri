2 · Analyze a video

⏱ Runs on a free CPU host — a clip takes ~1–3 min. First call after idle wakes the server (~30 s).


    

  


  
  

    
3 · Endpoints

    

      
Method / pathWhat it does

      
GET /healthLiveness — model & device loaded

      
GET /docsBuilt-in interactive test page

      
POST /analyzeform: video (file), pitch_length (float) → delivery JSON

    

    
Call it from code

    
curl -X POST "BASE/analyze" \
  -F "video=@clip.mp4" \
  -F "pitch_length=20.12"

    
Response fields

    

      
FieldMeaning

      
fps, total_frames, total_deliveriesClip metadata + deliveries detected

      
tracknum_points, average_confidence, physics_verdict, post_bounce_recovered

      
bounce / bounce_worldBounce in pixels / in real pitch metres

      
world_trajectoryBall path as [x_m, y_m, z_m] (lateral, down-pitch, height)

      
line / lengthBowling line & length + confidence

      
speed / speed_kmphEstimated pace in km/h

      
swing_cm / swing_typeSwing amount & direction (in/out/straight)

      
confidence_score, physically_validOverall trust in the delivery result
